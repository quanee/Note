Linux下使用命令行编译.c文件
	-vim hello.c
gcc命令基本格式
	-gcc [options] [filenames]
	options:
		-o:指定输出文件
		-E:预编译
		-S:编译为汇编语言
		-c:将汇编语言文件或C源文件编译为可重定向的目标文件(二进制形式),默认后缀为.o
	-gcc hello.c	//编译hello.c文件,默认输出到a.out文件
执行
	./a.out
-gcc hello.c -o hello-	// 编译hello.c文件,并输出到hello


crontab
-u user:指定用户
-e:编辑用户任务
-l:显示用户任务
-r:从/var/spool/cron目录中删除某个用户的crontab文件
-i:删除crontab文件时给予确认提示
文件格式:
	*	*	*	*	*	*
	|	|	|	|	|	|					
	分	时	日	月	周	命令
